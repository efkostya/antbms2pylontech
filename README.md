# ESP32 ANT BMS to Pylontech & MQTT Gateway

Умный шлюз на базе ESP32, который читает данные с аккумулятора под управлением **ANT BMS** (по UART), эмулирует заводской протокол **Pylontech RS485** для инверторов (Voltronic, Deye, Growatt, Solis и др.) и параллельно транслирует телеметрию в Умный Дом по **MQTT**.

## 🌟 Основные возможности

* **Полная эмуляция протокола Pylontech (V2.1):** Поддержка команд `42H`, `44H`, `47H`, `4FH`, `51H`, `92H`, `93H`, `96H`. Инвертор воспринимает самосборку как оригинальную батарею.
* **Аппаратная синхронизация защит:** Если ANT BMS отключает зарядный или разрядный MOSFET (например, по температуре или перекосу), шлюз мгновенно передает инвертору команду остановить ток.
* **Умное ограничение заряда:** Заряжайте LiFePO4 до безопасных 90-95% для продления срока службы. Лимит настраивается прямо в Web-интерфейсе.
* **Умное ограничение разряда:** При достижении нижнего порога SoC или напряжения ячейки - запрос инвертеру на принудительный заряд, независимо от его режима. В Web-интерфейсе кнопка ручного запроса для тестирования функции.
* **Учет энергии:** Интегрированный расчет пропущенных через батарею кВт·ч (заряд/разряд) на основе данных о мощности BMS. Оснащен алгоритмом защиты Flash-памяти ESP32 от износа.
* **Встроенное обновление по воздуху (Web OTA):** Возможность прошивки шлюза через встроенную веб-страницу без подключения USB-кабеля к компьютеру.
* **Полноценный Web-интерфейс:**
  * Живой мониторинг ячеек, температур, мощностей, статусов MOSFET и балансира.
  * Журнал (лог) входящих запросов по RS485 в реальном времени.
  * Изменение лимитов токов, напряжений и настроек MQTT без перепрошивки.
  * Автообновление страницы каждые 3 секунды.
* **MQTT Телеметрия:** Передача всех данных батареи (включая счетчики циклов, внутреннюю температуру ESP32 и статусы) в MQTT брокер.

## 🛠 Оборудование и подключение

Вам понадобятся:
1. Модуль **ESP32** (NodeMCU, WROOM и т.д.).
2. Модуль преобразователя **UART в RS485** (например, на базе MAX485) с поддержкой автоматического направления или выводом DE/RE.

3. Батарея с **ANT BMS**.

### Распиновка (по умолчанию в коде)

| ESP32 Pin | Подключение | Назначение |
| :--- | :--- | :--- |
| **RX 27** | ANT BMS TX | Чтение данных с BMS |
| **TX 26** | ANT BMS RX | Запрос данных у BMS |
| **RX 16** | RS485 RO | Прием запросов от инвертора |
| **TX 17** | RS485 DI | Отправка ответов инвертору |
| **Pin 4** | RS485 DE/RE | Пин управления направлением передачи RS485 |
| **GND** | Общая земля | Обязательно объедините GND ESP32, RS485 и BMS |

## 🚀 Установка и первый запуск

1. Откройте файл `ESP32_BMS_Gateway.ino` в Arduino IDE.
2. Установите необходимые библиотеки (если еще не установлены): `PubSubClient`, `Preferences`, `WebServer`.
3. В начале кода измените параметры Wi-Fi на свои:
   ```cpp
   const char* WIFI_SSID = "Ваш_Wi-Fi";
   const char* WIFI_PASSWORD = "Ваш_Пароль";
4. После прошивки - заходим на IP браузером. Его можно посмотреть в консоли порта ESP32 или в роутере Wifi.
